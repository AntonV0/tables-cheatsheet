Cheatsheet: Creating Tables Using HTML and CSS

In this cheatsheet we will cover several elements and sections used when adding tables to a webpage.

Table Elements

All tables contain these main elements:

Tag Name
<table> Table Container
<th> Table Header
<tr> Table Row
<td> Table Data

Table Sections

Tables are split into 3 sections - the table head, body, and foot.

Tag Name
<thead> Table Head
<tbody> Table Body
<tfoot> Table Foot
